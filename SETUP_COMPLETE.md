### Q: I'm stuck on a problem. What should I do?

**A**:

1. Spend 20-30 minutes thinking
2. Look at hints (not full solution)
3. If still stuck, read solution
4. Implement without looking
5. Redo after 2-3 days

### Q: How do I know if I understand?

**A**:

- Can explain to someone else?
- Can solve similar problems?
- Can code from scratch without reference?
- Understand time/space complexity?
